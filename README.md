# Mini Kanban Board

A modern Angular mini Kanban board with drag-and-drop, priority, filters, and undo/redo.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/Vignesh-184/minikanbanboard)

## Run locally

```bash
npm install
npm start
```

## Features

- 3 columns: Todo, In Progress, Done
- Add, edit, delete tasks
- Drag and drop between columns
- IndexedDB persistence
- Priority color coding
- Status filters
- Undo / redo actions
